# 📸 OpenCV Python - Part 2

Project ini berisi materi lanjutan OpenCV untuk pengolahan citra digital menggunakan Python.

## 📚 Materi yang Dibahas

### 1. Image Crop
- Crop menggunakan NumPy slicing
- Input koordinat dari user
- Crop menggunakan mouse event
- Simpan hasil crop

### 2. Image Resize
- Resize ukuran tetap
- Resize berdasarkan rasio
- Resize menggunakan fx & fy
- Interpolation:
  - INTER_NEAREST
  - INTER_LINEAR
  - INTER_CUBIC
  - INTER_AREA
  - INTER_LANCZOS4

### 3. Image Blending
- Menggabungkan 2 gambar (addWeighted)
- Blending beda ukuran (resize & fit window)
- Trackbar untuk kontrol alpha
- Overlay transparan

### 4. Color Conversion
- BGR → GRAY
- BGR → RGB
- BGR → BGRA (Alpha Channel)
- Manipulasi transparansi (Alpha)
